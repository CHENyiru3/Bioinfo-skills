# Tool Market

This directory is the inactive, repo-local source registry for selectable
bioinformatics tool bundles. Refs here are not part of an active section until
`bioinfo-sdd install-tool-bundle` copies a bundle into a section-local
`installed_refs/` revision.

The market is versioned with the repository, but it is outside SDD packs so
packs can stay focused on abstract workflow/task slots.
